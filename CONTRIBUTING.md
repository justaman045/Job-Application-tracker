# Contributing to North

Thank you for your interest! Here's how to contribute:

## Development

1. Clone the repo
2. Copy `.env.example` to `.env` and add your Firebase config
3. Run `npm install` then `npm run dev`

## Pull Requests

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/amazing`)
3. Make your changes
4. Run `npm run build` to verify
5. Submit a PR with a clear description

## Code Style

- ESLint and Prettier configs are included
- Run `npx prettier --write .` before committing
- Keep components focused and reusable
